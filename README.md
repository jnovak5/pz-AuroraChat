# AuroraChat

A feature-rich RP chat mod for **Project Zomboid Build 42** servers.

## Features

- **Proximity-based chat** — whisper, low, say, loud, shout volumes with configurable tile ranges
- **Language system** — 50+ languages with partial-understanding mechanics
- **Emote / Do / OOC** chat modifiers
- **Tape Recorder** item — record, play back, and distribute in-world messages
- **Typing indicator** — overhead and in-chat display
- **AFK system** — auto-detect and overhead indicator
- **Focus / Private chat** — filter messages or open a proximity private channel
- **Radio sync** — chat over radio frequencies
- **Admin tools** — admin hammer, chat override, language management
- **Character tools** — set name, name colour, hair/beard grow + colour, auto-clean, self-injury
- **Fully customisable** — per-player colour preferences, unread tab styles, keep-last-chat, and more

## Installation

Place the `AuroraChat` folder into your server's `mods/` directory and enable **AuroraChat** in the Mod Manager.  
All settings are available under **Sandbox Options → AuroraChat**.
